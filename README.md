## 🚀 Projetos em destaque

### MailWorks

[Repositório](https://github.com/igoralbuquerque12/MailWorks-API)

API multi-tenant para envio de e-mails construída com NestJS, Prisma, PostgreSQL e arquitetura serverless/event-driven na AWS.

O projeto suporta envio de e-mails transacionais, templates, campanhas em massa, códigos de autenticação e consulta de status de jobs. A infraestrutura utiliza API Gateway, AWS Lambda, Amazon SQS, Dead Letter Queue, Amazon SES/SMTP e CloudWatch para separar a requisição HTTP do processamento real de envio.

### Vibe Git + MCP Vibe Git

[Repositório Vibe Git](https://github.com/igoralbuquerque12/vibe-git)
[Repositório MCP Vibe Git](https://github.com/igoralbuquerque12/mcp-vibe-git)

Ferramenta para automatizar commits e Pull Requests seguindo padrões definidos de projeto e empresa.

O Vibe Git analisa alterações no repositório e gera mensagens de commit, descrições de PR e fluxos padronizados com base em regras como tipo de commit, idioma, template e convenções internas. Também desenvolvi um MCP para permitir que agentes de IA usem o Vibe Git como uma ferramenta controlada, reduzindo dependência de prompts longos e outputs imprevisíveis.

### Care Copilot

[Repositório](https://github.com/igoralbuquerque12/care-copilot)

Plataforma fullstack voltada para apoio médico durante atendimentos clínicos.

O sistema organiza pacientes, médicos, consultas, anamneses e histórico clínico, com funcionalidades de inteligência artificial para auxiliar na análise de informações do paciente, comparação com registros anteriores e apoio à tomada de decisão. A proposta é usar IA como suporte ao profissional de saúde, melhorando organização, contexto e produtividade durante o atendimento.

### Fala Comigo

[Backend](https://github.com/marcelomatheus/falacomigo-chat-server)
[Frontend](https://github.com/marcelomatheus/falacomigo-chat-web)

Projeto desenvolvido em parceria com [Marcelo Matheus](https://github.com/marcelomatheus).

Plataforma de chat em tempo real para prática de idiomas, conectando pessoas que querem aprender e conversar com outros usuários. A aplicação utiliza WebSocket para comunicação em tempo real e recursos de inteligência artificial para tradução, sugestões e correções. O backend utiliza NestJS, autenticação JWT, MongoDB via Prisma e Redis para filas e integração com Socket.IO.

### Movie Wish

[Repositório](https://github.com/igoralbuquerque12/movie-wish)

Aplicativo mobile desenvolvido em Flutter, conectado a um backend em NestJS.

O projeto permite descobrir filmes por meio de uma API externa, salvar títulos em uma lista pessoal e organizar filmes para assistir depois. Foi construído como uma aplicação fullstack simples, com foco em integração mobile-backend, consumo de APIs externas e organização de dados do usuário.

### PixelPhone

Projeto privado / freelance

Sistema de discador e gestão de leads desenvolvido para apoiar equipes que realizam muitas ligações diárias.

A plataforma ajuda a organizar contatos, controlar tentativas de ligação, acompanhar status dos leads e gerenciar follow-ups. O objetivo é tornar o processo comercial mais rastreável, produtivo e organizado para secretárias e equipes de atendimento.
